## 简介

kafka是一个实时数据管道和流应用.特点:水平扩展,容错,极为迅速

3个关键能力:

1. 发布和订阅记录流
2. 以一种容错和持久化方式来存储记录流
3. 记录实时流

4个核心APIs

1. Producer API
2. Consumer API
3. Streams API
4. Connector API
